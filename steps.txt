Step 1. Create a folder called .github and inside it we shall create a subfolder called workflows
Step 2: Create a .yaml which will contain the workflow
Step3: In the workflow;
- Assign a workflow name
-Specify a trigger on a branch
- Add a runner 
- Create a job with the following steps
        - Checkout out code using prebuilt code actions
        - Set up node on the runner with a prebuilt actions
        - Run tests





name: my workflow                       #workflow name

on:                                     #event(s) that trigger this workflow
    push:
        branches: [main, develop]
    
jobs:                                   #one or more jobs
    my-job:                             #job id or job name
        runs-on: ubuntu-latest          #runner os
        steps:                          #sequential steps
            - name: checkout code
              uses: actions/checkout@v4 #reusable actions
            - name: say hello
              run: echo "helloworld"    #shell command
